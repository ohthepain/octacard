# Auditioning Samples from Packs in the Cloud

A global sample is a sample that exists in a pack on the server, not in a local folder

Global samples are downloaded into a temporary area that is cleared out when ... [?]

Drag a sample from global to local folder - put sample right into target folder
Drag a sample from global to local pack - warn - are you sure you want to edit this pack?
Drag a sample from global to stack block
Drag a pack from global to local
Drag a pack from global to stack block - why not!

The user can audition a global sample in the following UX flows:
- tap ‘play’ button next to the sample in the filepane. there is a play button next to each sample in the filepane. When playing the play button turns into a stop button. 
- tap the sample. This will open the wave view and start downloading the sample into the temp files. Once it’s finished downloading the user can play the sample just like a sample on their local folders.
- drag the sample from the filepane into a stack block
