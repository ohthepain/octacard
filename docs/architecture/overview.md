# System Architecture

Octacard is a sample manager and convert. It lets users upload folders of sample files as 'packs'.

## Overview

The system consists of 
- tanstack start front and back end
- hono back end
- tailwindcss
- better-auth with Google OAuth
- bullmq for jobs
- essentia and CLAP for audio analysis
